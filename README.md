# NicolasKa_12_10072022 : SportSee project

## Before starting the frontend 
Clone the backend repository : `https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard.git`
and be sure to follow the instructions provided on the repo.

## To start the frontend project 
```
install deps : 
yarn

start in dev mode :
yarn dev
```

## To build the project for production
`yarn build` the built project will be accessible in the `dist` folder.

## If you want to see the built app
`yarn preview`
